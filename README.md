#### MobileTechStackAnalyzer - приложение для анализа стека использованных инструментов в приложениях, установленных на Android устройстве.

Оно определяет языки программирования, фреймфорки, использованные библиотеки, разрешения и предоставляет техническую информацию о сборке приложения.

##### Возможности детального анализа:
- Определение фреймворка (Native Android, Flutter, React Native, Xamarin, Cordova/Ionic, Unity, Kotlin Multiplatform, NativeScript)
- Определение использованных языков (Kotlin, Java, C/C++, JavaScript, Dart, C#, Python)
- Определение использования сторонних библиотек с категоризацией по виду деятельности
- Определение нативных библиотек
- Определение размера APK и поддерживаемой архитектуры (32/64-bit)

Использованные технологии:
- Kotlin 
- DexLib2
- Timber
- GitHub Actions
